## üîß Generate NEXTAUTH_SECRET

**Mac/Linux:**
```bash
openssl rand -base64 32
```

**Windows PowerShell:**
```powershell
[Convert]::ToBase64String((1..32 | ForEach-Object { Get-Random -Maximum 256 }))
```

**Online:**
[generate-secret.vercel.app/32](https://generate-secret.vercel.app/32)
